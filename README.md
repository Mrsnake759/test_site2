# Альфа — витрина бренда на Nuxt 3 (SSG)

Сайт подготовлен для генерации статических файлов и загрузки на обычный хостинг (Reg.ru, `public_html`).

## Локальный запуск
```bash
npm i
npm run dev
```

## Сборка статической версии
```bash
npm run build
npm run generate
```

После `npm run generate` готовая статика находится в `.output/public`.

## Деплой на Reg.ru
1. Локально выполните:
   ```bash
   npm i
   npm run generate
   ```
2. Откройте папку `.output/public`.
3. Скопируйте её содержимое в `public_html` на хостинге Reg.ru.
4. Проверьте, что `public_html/index.html` существует.

## Страницы
- `/` — главная
- `/contacts` — контакты (полноценная страница)
- `/about`, `/catalog`, `/services`, `/delivery` — заглушки, подготовлены для следующих итераций.

## Важное
Все формы сейчас работают в демо-режиме: показывают сообщение «Заявка отправлена (демо)» и не отправляют данные на сервер.
